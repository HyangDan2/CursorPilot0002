# Multimedia Player with Interleave Mixing

## Features
- Play, Stop, Pause, Loop video playback
- Load and mix two videos (interleaved mixing)
- Display FPS
- Save mixed video
- PySide6 GUI, OOP structure

## Requirements
- Python 3.8+
- PySide6
- OpenCV (opencv-python)

## How to run
```
python main.py
```
