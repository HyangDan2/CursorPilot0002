# CursorPilot0002

## 주요 기능
- PySide6 기반 패턴/이미지 디스플레이 및 관리 GUI
- RGBW 패턴 및 이미지 표시, 밝기(그레이레벨) 조절
- Pattern Manager로 이미지 추가/관리 (Image 폴더)
- Tab 키로 오버레이(메타데이터, 마우스 위치 RGB) 표시
- ESC로 풀스크린 토글 및 geometry 복원, 메뉴바 자동 숨김/복원
- 마지막 이미지 자동 복원(Ctrl+I)

## 키 바인드
- 1/2/3/4 : Red/Green/Blue/White 패턴 (밝기 255로 초기화)
- ↑/↓ :
  - RGBW 패턴: 밝기 1단계(0~255) 증감
  - 이미지: 밝기 1단계(0~100%) 증감
- Ctrl + + / Ctrl + - :
  - RGBW 패턴: 밝기 32단계 증감
  - 이미지: 밝기 10단계(10%) 증감 (경계구간 보정)
- Tab : 오버레이(메타데이터+RGB) 토글
- ESC : 풀스크린 토글 (메뉴바 자동 숨김/복원)
- Ctrl+I : 마지막 이미지 복원

## Pattern Manager
- 이미지 추가 시 Image 폴더에 복사 저장
- 실행 시 Image 폴더 내 파일 리스트 자동 표시
- 더블클릭 시 메인 윈도우에 이미지 표시

## 오버레이
- Tab 키로 좌상단에 오버레이 표시
- 파일명, 밝기 레벨, 마우스 위치 RGB값 표시

## 밝기 조절
- RGBW 패턴: 0~255(1단계), 32단계 점프
- 이미지: 0~100%(1단계), 10단계(10%) 점프, 경계구간 보정
